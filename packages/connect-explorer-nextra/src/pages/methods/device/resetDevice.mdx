## Reset device

Perform device setup and generate new seed.

```javascript
const result = await TrezorConnect.resetDevice(params);
```

### Params

<CommonParamsLink />

#### ResetDevice

<ParamsTable schema={ResetDevice} descriptions={paramDescriptions} />

### Example

```javascript
TrezorConnect.resetDevice({
    label: 'My fancy Trezor',
});
```

### Result

Success type

```javascript
{
    success: true,
    payload: {
        message: 'Device successfully initialized'
    }
}
```

Error

```javascript
{
    success: false,
    payload: {
        error: string // error message
    }
}
```
