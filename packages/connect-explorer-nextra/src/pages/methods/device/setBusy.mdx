## Set busy

<Callout type="error" emoji="️🚫">
    **Management command** - this method is restricted to Trezor.io and can't be used in 3rd party
    applications.
</Callout>

Show a "Do not disconnect" dialog instead of the standard homescreen.

```javascript
const result = await TrezorConnect.setBusy(params);
```

<Callout type="info">**Supported only by T2T1 with Firmware 2.5.3 or higher!**</Callout>
